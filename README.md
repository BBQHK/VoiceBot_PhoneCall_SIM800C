# PhoneCallSIM800C
It implements the Phone Call using ATK-SIM800C GSM/GPRS Module. 
Moreover, Text2Speech and Speech2Text will be integrated. 
It deals with multiple scope, like serial programming, audio processing, and OOP knowledge.
More details will be available later.

## Setting up a Python development environment
Always remember to use a pre-project environment when developing locally with Python. Please refer to Google Cloud Guide (https://cloud.google.com/python/docs/setup#windows) to set up virtual environment.
